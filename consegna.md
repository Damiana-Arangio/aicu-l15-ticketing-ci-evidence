# Consegna studenti - Lezione 15

## Obiettivo

Proteggere con Playwright un comportamento visibile della dashboard.

## Passi

1. Verifica la baseline:

   ```bash
   pnpm test
   pnpm test:e2e
   ```

2. Scegli un comportamento visibile.
3. Definisci azione, risultato e controllo.
4. Completa o modifica un solo test in `tests/e2e`.
5. Usa locator basati su role, label o contesto visibile.
6. Esegui il test.
7. Leggi l'output prima di correggere eventuali errori.

Puoi partire da una combinazione `urgencyLabel`, ma puoi proteggere qualsiasi comportamento chiaro della dashboard.

## Pronto quando

Puoi mostrare:

```txt
file
comando
output
comportamento protetto
```

Non serve compilare un report separato.
